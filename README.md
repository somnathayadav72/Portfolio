# Somnath Yadav — Interface Fieldbook

A frontend-only Next.js portfolio for Somnath Yadav, built around the working
systems behind complex product interfaces: workflows, data, responsiveness,
performance, and visual experiments.

## Quick start

```bash
npm install
npm run dev
npm run lint
npm run validate:assets
npm run build
```

The site is intentionally frontend-only. The contact form prepares a local
email handoff; no API, database, authentication, analytics, or server action is
connected.
